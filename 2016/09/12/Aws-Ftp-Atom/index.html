AWS-EC2 + Vsftpd + Atom

  • 2016-09-12

Convenience way for working with service side project。 For sake of simplicity，this tutorial will only lead you major steps。

OS Environment

Mac OS X EI Capitan
AWS-EC2 Ubuntu 14.04.4 LTS

1. Setup Atom

  • Install Atom

  • Install Atom packages remote-ftp

2. Setup AWS-EC2

  • Complete tutorial AWS

3. Setup AWS Security Group

  • Add new Custom TCP Rule with protocol TCP and port range 20 - 21
  • Add new Custom TCP Rule with protocol TCP and port range 13000 - 13100

4. Setup vsftpd

Connect to AWS-EC2 via ssh

  • Install vsftpd

  • Modify vsftpd configuration

  • Change few settings within vsftpd.conf

  • Restart vsftpd

  • Create new group

  • Create new user and add into your new group

  • Set password for new user

  • Create ftp home directory

  • Change permission

  • Change owner

5. Final

  • Combine AWS-EC2 + vsftpd + Atom

  • Ftp your ip and check wether is correct or not?

  • Follow the getting started section

  • Finish
