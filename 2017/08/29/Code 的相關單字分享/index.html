Code 的相關單字分享

  • 2017-08-29

紀錄平常觀看技術文檔發現覺得不錯的單字

Spaghetti code

就是貶抑你的 code 寫得很沒有結構，就像義大麵這般的亂

Pros and Cons

優點跟缺點的簡稱

Statically typed programming languages

就是我們常說的 強型別語言，必須在 compiler time 的時候就給予型態，比如 C，Java，Objective-C。

Dynamically typed programming languages

就是我們常說的 弱型別語言，比如 php，Haskell。

Opt-in

Opt-in is the short form of “opting in.” It means that someone wants to be involved in something. In the age of technology and the world of sales, the term means that someone has given you permission to send him emails.

簡單來說，歡迎寄信給他，他給你許可了。

Boilerplate code

“Boilerplate code” is any seemingly repetitive code that shows up again and again in order to get some result that seems like it ought to be much simpler.

就是經常重複出現的代碼。

Stringly Typed

Method parameters that take strings when other more appropriate types should be used

就是不該使用 string 的時後卻使用 string，只為了求快或者沒考慮到後續維護

nitpicking

雞蛋裡挑骨頭的意思
