Swift 3.0 Unsafe World

Feb 11, 2017

最近在研究 Reachability 的時候，發現了許多 Unsafe 開頭的 API, 然後在網路上找到這篇，覺得寫得不錯，於是決定來把它翻譯並且搞懂 Unsafe World。

廢話不多說，因為文章只討論 Swift 3.0，並且提供 C 與 Swift 兩種對照寫法！而使用 Unsafe 的目的當然就是為了跟 C 溝通，所以當然要提供兩種版本。
